# NetClimat - Website Development Project

## Overview
Professional website developed for NetClimat, a ventilation and air conditioning cleaning service company based in Quebec.

The objective was to create a modern, responsive and SEO-friendly website to improve the company's online presence, showcase services and facilitate customer contact.

## Client
NetClimat - Ventilation Cleaning Services

## My Role
Web Developer / Full Stack Developer

Responsibilities:
- Designed and developed the complete website
- Created responsive UI/UX design
- Integrated service pages and contact sections
- Optimized website performance and user experience
- Configured deployment and hosting environment

## Features
✅ Responsive design (Desktop, Tablet, Mobile)  
✅ Modern landing page  
✅ Service presentation pages  
✅ Contact form integration  
✅ SEO optimization  
✅ Marketing sections and promotions  
✅ Performance optimization
